參數 α (Alpha) 2.0
參數 β (Beta) 2.0
期望值 E[θ]
0.500
= α / (α + β)
眾數 Mode
0.500
= (α-1) / (α+β-2)

0.050

📚 Beta 分佈的直覺理解

參數的意義：

  • α：可以想成「成功」的次數 + 1
  • β：可以想成「失敗」的次數 + 1

常見的 Beta 分佈形狀：

Beta(1, 1) = 均勻分佈

完全無知，所有機率都同樣可能。

📊 平坦的直線

Beta(2, 2) = 偏向中間

相信機率接近 0.5（公平）。

📊 中間高、兩邊低的鐘形

Beta(10, 2) = 偏向高機率

相信成功機率較高。

📊 右偏分佈

Beta(2, 10) = 偏向低機率

相信成功機率較低。

📊 左偏分佈

Beta(0.5, 0.5) = U 型

相信極端值（0 或 1）。

📊 兩端高、中間低

Beta(50, 50) = 尖峰

非常確定機率是 0.5。

📊 非常集中的尖峰

💡 為什麼 Beta(2,2) 適合當硬幣的先驗？

  • 期望值 = 2/(2+2) = 0.5，表示我們相信硬幣是公平的
  • 但變異數不為 0，表示我們保持開放態度
  • 相當於「之前丟過 1 次正面、1 次反面」的經驗
  • 不會太強勢（像 Beta(100,100)），也不會完全無知（像 Beta(1,1)）

🎯 在 MAP 估計中的應用

假設我們丟硬幣 10 次，全部都是正面：

MLE（沒有先驗）：

θ̂MLE = 10/10 = 1.0 ← 過度自信！

MAP（使用 Beta(2,2) 先驗）：

θ̂MAP = (10+2-1)/(10+2+2-2) = 11/12 ≈ 0.917 ← 較合理！

先驗分佈提供了「溫和的懷疑」，避免極端估計。

貝氏更新公式：

後驗 = Beta(α + 成功次數, β + 失敗次數)

這就是為什麼 Beta 分佈是「共軛先驗」：後驗也是 Beta 分佈！
